Diretivas do projeto

  1. Configure um novo projeto diferente do que criamos nas aulas de introdução.
  2. Crie uma página de Listagem de Produtos buscando os dados da API que vocês criaram nas aulas de Spring.
    • Você precisa exibir na listagem de produtos em uma tabela os campos: Nome, Código de Barras, Preço e uma coluna de Ações.
    • Necessário ter dois botões na coluna de ações: Editar e Excluir.
    • Editar: Encaminha para a página de edição
    • Excluir: Exibe uma confirmação, caso o usuário confirme, realizar a remoção.
  3. Crie uma página de Cadastro de Produtos com os campos:
    • Nome do Produto (string)
    • Código de Barras (string)
    • Preço (number)
  4. Crie uma página de Edição de Produtos com os campos:
    • Nome do Produto (string)
    • Código de Barras (string)
    • Preço (number)
  5. Seguir as seguintes diretrizes:
    • Objeto de Produto tipado com interface/type (sem uso de any)
    • Validações no cadastro/edição do tamanho do campo, obrigatório, tamanho do campo, tipo número (campo Preço)
